HTML Attributes

Lang Attribute

<!DOCTYPE html> <html lang="en-US"> <body> <h1>My First Heading</h1> <p>My first paragraph.</p> </body> </html>

Title Attribute

<!DOCTYPE html> <html lang="en-US"> <body> <p title="Web Development WOrkshop"> Welcome to Web Development Workshop. You are going to learn Basic HTMl and CSS. </p> </body> </html>

Href Attribute

<!DOCTYPE html> <html lang="en-US"> <body> <a href="http://www.google.com">This is a link</a> </body> </html>

Alt Attribute

<!DOCTYPE html> <html lang="en-US"> <body> <img src="http://www.telegraphindia.com/1090820/images/2008OT.jpg" alt="IEM Salt lake"/> </body> </html>

ID Attribute

<!DOCTYPE html> <html lang="en-US"> <body> <div id="Hello"> Welcome to Hello World! </div> <div id="World"> Welcome to Hello World! </div> </body> </html>

Class Attribute

<!DOCTYPE html> <html lang="en-US"> <body> <div class="Hello-World"> Welcome to Hello World! </div> <div class="Hello-World"> Welcome to Hello World! </div> </body> </html>

Style Attribute

<!DOCTYPE html> <html lang="en-US"> <body> <div style="color:green;"> Welcome to Hello World! </div> <div style="font-size:18px"> Welcome to Hello World! </div> </body> </html>
