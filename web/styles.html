HTML Styles

HTML Inline Style

<!DOCTYPE html> <html lang="en-US"> <body> <p style="color:red;"> This paragraph contains a lot of lines in the source code, but the browser ignores it. </p> <p style="color:green;"> This paragraph contains a lot of spaces in the source code, but the browser ignores it. </p> <p style="color:blue;"> The number of lines in a paragraph depends on the size of the browser window. If you resize the browser window, the number of lines in this paragraph will change. </p> </body> </html>

HTML Internal Style

<!DOCTYPE html> <html lang="en-US"> <style> #p1{ color: red; } #p2{ color: green; } #p3{ color: blue; } </style> <body> <p id="p1"> This paragraph contains a lot of lines in the source code, but the browser ignores it. </p> <p id="p2"> This paragraph contains a lot of spaces in the source code, but the browser ignores it. </p> <p id="p3"> The number of lines in a paragraph depends on the size of the browser window. If you resize the browser window, the number of lines in this paragraph will change. </p> </body> </html>

If you resize the browser window, the number of lines in this paragraph will change. </p> </body> </html>
