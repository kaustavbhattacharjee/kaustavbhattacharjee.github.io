Bold or Strong Text Formatting

<!DOCTYPE html> <html lang="en-US"> <body> <p>I am written in <b>Bold</b></p> <p>I am written in <strong>Strong</strong></p> </body> </html>

Italic oe Emphasized Text Formatting

<!DOCTYPE html> <html lang="en-US"> <body> <p>This text is normal.</p> <p>This text is <i>italic</i>.</p> <p>This text is <em>emphasized</em>.</p> </body> </html>

Small Text Formatting

<!DOCTYPE html> <html lang="en-US"> <body> <h2>This text is <small>small</small>.</h2> </body> </html>

Marked Text Formatting

<!DOCTYPE html> <html lang="en-US"> <body> <p>I am <mark>Marked</mark> out</p> </body> </html>

Delete Text Formatting

<!DOCTYPE html> <html lang="en-US"> <body> <p>I am a <del>bad</del> good human being.</p> </body> </html>

Inserted Text Formatting

<!DOCTYPE html> <html lang="en-US"> <body> <p>I am a <ins>good</ins> human being.</p> </body> </html>

Subscripted Text Formatting

<!DOCTYPE html> <html lang="en-US"> <body> <p>H<sub>2</sub> + O<sub>2</sub> = H<sub>2</sub>O</p> </body> </html>

Supercripted Text Formatting

<!DOCTYPE html> <html lang="en-US"> <body> <p>a<sup>2</sup> + b<sup>2</sup> = c<sup>2</sup></p> </body> </html>
